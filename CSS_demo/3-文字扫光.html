CSS3 background-clip 属性 
    - 规定背景的绘制区域
    - 值：
        + border-box    背景被裁剪到边框盒（默认）
        + padding-box   背景被裁剪到内边距框。
        + content-box   背景被裁剪到内容框。
        + text          背景被裁剪为文字的前景色
